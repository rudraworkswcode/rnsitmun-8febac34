# Resource Images
## RNSIT Model United Nations

This folder contains images and visual assets for the Resources page.

## Folder Structure

```
images/
├── beginners-guide/          # Images for Beginner's Guide section
├── research-preparation/     # Images for Research & Preparation section
├── committee-resources/      # Images for Committee Resources section
└── skills-development/       # Images for Skills Development section
```

## Usage Guidelines

### Folder Purposes:

**beginners-guide/**
- Images for "What is Model UN?" guide
- First-time delegate resources
- Introductory materials

**research-preparation/**
- Research template screenshots
- Position paper examples
- Note-taking guides

**committee-resources/**
- Committee-specific guides (UNHRC, UNSC, etc.)
- Background guide covers
- Committee comparison charts

**skills-development/**
- Public speaking visuals
- Negotiation strategy diagrams
- Resolution writing examples

## File Naming Convention

Use descriptive, lowercase names with hyphens:
- ✅ `public-speaking-tips.jpg`
- ✅ `unhrc-background-guide.png`
- ✅ `position-paper-template.jpg`
- ❌ `image1.jpg`
- ❌ `Photo 2.png`

## Image Specifications

**Recommended:**
- Format: JPG or PNG
- Max size: 2MB per image
- Dimensions: 1200x800px or larger (maintain aspect ratio)
- Compression: Use tools like TinyPNG for optimization

**For Icons/Badges:**
- Format: PNG (with transparency) or SVG
- Size: 512x512px
- Transparent background preferred

## Adding New Images

1. Place images in appropriate category folder
2. Use descriptive file names
3. Optimize images before uploading
4. Update Resources.tsx to reference new images
5. Test on both desktop and mobile

## Current Structure

This folder structure is ready to receive:
- Screenshot images from the user
- Custom icons and badges
- Background images
- Thumbnail previews
