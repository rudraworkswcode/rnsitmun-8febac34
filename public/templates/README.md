# MUN Delegate Templates
## RNSIT Model United Nations

This folder contains downloadable templates to help delegates prepare for Model UN conferences.

## Available Templates

### 1. Position Paper Template
**File:** `position-paper-template.md`

A comprehensive guide for writing position papers including:
- Header format
- Country profile structure
- Historical background sections
- Policy positions
- Proposed solutions
- Citation guidelines

### 2. Resolution Template
**File:** `resolution-template.md`

Complete resolution writing guide with:
- Proper formatting rules
- Preamble and operative clauses
- Examples of both formats
- List of common preambulatory and operative phrases
- Tips for writing strong resolutions

### 3. Research Notes Template
**File:** `research-notes-template.md`

Organized template for research including:
- Country profile sections
- Topic analysis framework
- UN action tracking
- Argument preparation
- Negotiation strategy planning
- Conference notes section

### 4. Delegate Preparation Checklist
**File:** `delegate-preparation-checklist.md`

Complete preparation checklist covering:
- Research phase (4-6 weeks before)
- Writing phase (2-3 weeks before)
- Skills development
- Strategy planning
- Materials preparation
- Conference day tips

## How to Use

1. **Download** the template you need
2. **Open** in any text editor or word processor
3. **Fill in** your information following the guidelines
4. **Customize** as needed for your specific committee/conference
5. **Save** and submit according to your conference requirements

## Format Notes

- Templates are provided in Markdown (.md) format
- Can be opened in any text editor (Notepad, TextEdit, VS Code, etc.)
- Can be converted to PDF or Word format using online converters
- Preserve formatting when copying to other documents

## Tips for Best Results

✅ Start early - don't wait until the last minute
✅ Follow the structure but customize for your needs
✅ Always cite your sources properly
✅ Proofread before submitting
✅ Save multiple versions as you work
✅ Ask for feedback from advisors/mentors
